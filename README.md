# 🦠 Virus 后端项目

该项目为 `Virus` 系统的后端部分，基于 Maven 构建，包含核心业务逻辑与应用启动类。

---

## 📥 克隆项目

```bash
git clone https://github.com/Allmlhy/Virus.git
cd Virus
```

## 🚀 直接定位到
```bash
target/classes/com/dhr/maven/virus_backend/VirusBackendApplication.class
```

### 点击运行
```bash
右键点击 VirusBackendApplication → 选择 Run 运行。
```
